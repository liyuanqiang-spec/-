# 马来西亚游客扫码购微信小程序原型

这是一个面向“马来西亚游客扫码购买、中国国内供应链发货”的微信小程序前端原型。游客可通过门店/导游/展台二维码进入小程序，浏览商品、查看跨境发货说明，并提交收货与联系方式形成订单草稿。

## 功能范围

- 扫码入口：支持 `scene` 参数解析，记录渠道、门店/导游码。
- 商品浏览：展示适合游客回购的爆品、价格、库存、发货地与时效。
- 商品详情：突出正品保障、跨境运输、售后与支付说明。
- 下单表单：采集游客姓名、手机号/WhatsApp、马来西亚收货地址、护照/证件尾号（可选）、备注。
- 本地订单草稿：无后端时将订单保存在本地缓存，便于后续对接微信支付、ERP、物流与客服。

## 目录结构

```text
app.js / app.json / app.wxss       小程序全局配置与样式
pages/home                         扫码落地页与业务说明
pages/products                     商品列表页
pages/product-detail               商品详情页
pages/order                        下单页
utils/catalog.js                   示例商品数据
utils/scene.js                     二维码 scene 参数解析
```
