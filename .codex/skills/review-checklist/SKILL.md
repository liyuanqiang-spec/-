# review-checklist

## Purpose

Provide a fast acceptance check after each worker result.

## Checklist

- Task matched
- Files updated
- Status visible
- Tests or checks ran
- No unrelated rewrite
- Next action clear

## Output

Return one of: PASS, NEEDS_FIX, BLOCKED.
