# docx-pdf-writer

## Purpose

Prepare structured Chinese documents with clear title, background, problems, suggestions, and implementation path.

## Rules

- Keep Chinese formatting clean.
- Avoid broken numbering and stray Latin markers.
- Separate proposal, research report, memo, and implementation plan styles.
- When creating real `.docx` or `.pdf` artifacts, follow the local document/PDF skill instructions first.

## Output

- Markdown draft first when useful.
- Then generate document artifact when requested.
