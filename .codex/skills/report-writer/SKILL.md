# Report Writer Skill

Use this skill when producing research or operational reports.

## Rules

1. Start with the conclusion.
2. Separate facts, assumptions, and recommendations.
3. Include file paths and test results.
4. Keep the report practical and short unless the user asks for detail.
5. Update `STATUS.md` after each finished phase.

## Output

- Conclusion
- Completed work
- Files changed
- Test results
- Current issues
- Next step
