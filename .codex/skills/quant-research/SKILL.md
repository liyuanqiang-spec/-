# Quant Research Skill

Use this skill before building or changing a quant strategy.

## Rules

1. Start with a short written plan before implementation.
2. Define the hypothesis, market, instrument, time range, data fields, and expected signal direction.
3. Separate research, backtest, simulation, and real execution. Real execution is not allowed in this project.
4. Record assumptions and known weaknesses in `STATUS.md`.

## Output

- Research question
- Data required
- Validation method
- Risk notes
- Next task
